Docker Deployment

To deploy this application using Docker:

# Clone the repository
git clone <repository-url>
cd smartfileguardian

# Build and start all services
docker-compose up -d

# View logs
docker-compose logs -f

# Scale workers (optional)
docker-compose up -d --scale worker=3

# Stop services
docker-compose down
The Docker setup includes PostgreSQL database, Redis for caching, Celery workers for background processing, and Nginx as reverse proxy.
